1. Apresente 5 sistemas operacionais suportados pelo Raspberry Pi, e algumas de suas características.
1. Raspbian
É uma versão do Debian permitindo ao usuário navegação na internet e ferramentas de configuração do computador.
2. Ubuntu
O Ubuntu contém aplicativos embarcados, repositórios onde é possível baixar outras ferramentas se necessário.
Uma versão mais adequada ao Raspberry Pi 2 e 3 é o Ubuntu Mate.
3. OSMC
É um sistema operacional (open source media center) baseado no Debian e Kodi Media Center. Este sistema permite a organização e reprodução
de vídeos, visualização de fotografias, interface simples e é um sistema gratuito.
4. Recalbox
Sistema para emular jogos.
5. Pidora
Desenvolvido pelo Projeto Fedora, o Pidora processa para Raspberry Pi 2 e 3 e permite configurar o sistema sem monitor. É possível
acessar pela rede de outro computador.

2. Apresente as formas de instalação de sistemas operacionais para o Raspberry Pi.
É possível instalar pelo Noobs ou baixar a imagem do Raspbian no site desenvolvedor do Raspberry e gravá-la em um cartão sd. Neste último método, o Raspbian
vem pré-configurado, dessa forma, só é preciso conectar os periféricos, teclado, mouse e monitor.

3. Apresente 3 formas de desenvolvimento de software para o Raspberry Pi.
Um software pode ser desenvolvido em várias linguagens de programação. A mais utilizada é a Python, mas também outras como Java, Javascript, linguagem C, C++ podem ser utilizadas
dependendo da aplicação. 

4. Apresente 3 formas de acesso remoto ao Raspberry Pi.
SSH - protocolo que permite utilizar o Raspberry através de outro computador.
TightVNC - derivado do VNC Viewer que permite a visualização também através de outro computador.
VNC Viewer - acesso remoto em MAC, Windows e Linux.
